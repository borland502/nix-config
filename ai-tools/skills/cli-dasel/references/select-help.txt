Select properties from the given file.

Usage:
  dasel -f <file> -r <json,yaml,toml,xml,csv> <selector> [flags]
  dasel [command]

Available Commands:
  completion  Generate the autocompletion script for the specified shell
  delete      Delete properties from the given file.
  help        Help about any command
  man         Generate manual pages for all dasel subcommands
  put         Write properties to the given file.
  validate    Validate a list of files.

Flags:
      --colour                   Print colourised output.
      --csv-comma string         Comma separator to use when working with csv files. (default ",")
      --csv-comment string       Comma separator used when reading csv files.
      --csv-crlf                 True to use CRLF when writing CSV files.
      --csv-write-comma string   Comma separator used when writing csv files. Overrides csv-comma when writing.
      --escape-html              Escape HTML tags when writing output.
  -f, --file string              The file to query.
  -h, --help                     help for dasel
      --indent int               The indention level when writing files. (default 2)
      --pretty                   Pretty print the output. (default true)
  -r, --read string              The parser to use when reading.
  -s, --selector string          The selector to use when querying the data structure.
  -v, --version                  version for dasel
  -w, --write string             The parser to use when writing. Defaults to the read parser if not provided.

Use "dasel [command] --help" for more information about a command.
